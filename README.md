# Image viewer

A ppm image viewer made in C. This project was made following [This Video](https://www.youtube.com/watch?v=sItRLFjbqvo).
This image viewer can only open .ppm files.

## How to run
Run `build.sh` to compile the project and to run it use this command:

```bash
cat image.ppm | ./build/image_viewer
```
